# glossary_tkinter
一个用于搜索和保存单词的应用
----
#### 使用Python3内置tkinter界面库
#### 使用requests模块搜索爬取单词
#### 使用sqlite3数据库保存本地词库
#### 支持多线程导入保存单词
#### 支持自定义单词分类
#### 支持自定义词义高亮显示
##### 更新中...
